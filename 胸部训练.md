# 锁定肩胛骨
- 大部分人控制肩胛骨下压的肌群比较薄弱，容易驼背耸肩溜肩
- 肩胛骨后缩，尝试用背部夹住手指
- 肩胛骨后缩下压，腋窝处有酸胀感
- 有难度的是手臂活动的情况下依旧锁定肩胛骨
- 需要在手臂伸直时肩胛骨依旧是收紧的状态，不要把肩送出去
- 平躺时可以往下挪动背部，借助摩擦力让肩胛骨收的更紧

# 哑铃卧推
- 锁定肩胛骨，收下巴
- 平躺时可以往下挪动背部，借助摩擦力让肩胛骨收的更紧
- 挪动背部时会起桥，程度不要太多，一个手掌高度就好
- 身体稳定，核心绷紧，腹式呼吸，臀腿稳定
- 下放时拳头和乳头在一条水平线上，前臂始终垂直地面
- 手臂尽可能往下放，拉开胸部肌纤维
- 手肘有意识往内夹

# 上斜哑铃卧推
- 坐凳可以稍微往上翘
- 挪动上背部，收紧肩胛骨
- 上斜卧推不需要起桥
- 下方时拳头和锁骨在一条水平线上
- 由于胸上束纤维力量较小，上斜卧推重量一般比平板卧推重量小

# 为什么推荐做哑铃卧推，而不是杠铃卧推
- 因为哑铃卧推下放幅度和内夹幅度更大，更能练到胸肌外侧和内侧（这两侧无法通过分别的动作来孤立训练，因为其功能相同）
- 哑铃卧推还能改善左右胸大小力量不均衡的问题
- 夹胸这个动作能练到胸的内中外3侧，但是难上大重量，所以一般在最后力竭时做补充训练
- 杠铃的优势是冲大力量，让新手迅速的积累力量，后期突破瓶颈期很有用
