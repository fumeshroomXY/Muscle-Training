# 肌肉发力顺序
- 大部分躯干上的肌群，靠近身体的肌肉要先发力
- 比如坐姿划船的发力顺序：
  - 手臂先不动，斜方肌发力，将肩胛骨往后缩，肩胛骨带动肩部运动
  - 缩到一定程度再背阔肌发力，将手肘往后拉，肩部带动手肘运动
  - 最后手肘往后移动，将重量拉近身体
- 如果顺序不对，大部分做功距离让手臂承担了，背部就找不到感觉了
- 比如哑铃卧推
  - 正确顺序：胸肌优先收缩，带动手肘窝往中间靠拢，再顺带把手臂伸直
  - 先把手臂伸直，再把胸肌往内夹，就很容易出现胸肌找不到收缩感的情况

- 例外：少部分四肢肌肉动作，远离身体的肌肉先发力
  - 比如面拉，要三角肌后束先发力，再是斜方肌发力，夹紧肩胛骨
