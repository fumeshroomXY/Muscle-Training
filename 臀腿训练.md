# 屈髋
- 屈髋：身体向前折叠，头和脚靠近；伸髋：身体向后伸展，头和脚靠近
- 保持小腿始终垂直地面
- 往后推臀部，不是撅屁股
- 身体自然向前俯身，背部挺直，但是不要反弓，眼睛看着地板
- 直到大腿后部很紧，这就是目前屈髋的极限
- 伸髋，将髋部往前顶，同时夹紧屁股


# 徒手深蹲
- 站距1.5倍肩宽（可个人调整），脚尖微微外八（站距越宽外八幅度越大）
- 保持重心在足弓中间，根据个人情况膝盖可以超过脚尖
- 背部挺直，但是不要反弓
- 膝盖朝向和脚尖方向一致，甚至可以更朝外一点，不能内扣
- 蹲至大腿和地面平行

# 杠铃深蹲
- 用自由深蹲架更好，训练身体整体稳定性
- 起始高度不要超过锁骨位置
- 握杆位置，小拇指贴左线，远到可以大拇指贴右线
- 肩胛带扭矩（将肩胛骨往内旋，往下压收紧，手臂不会轻易晃动），将杠铃放在斜方肌上
- 窄站距，绷紧核心，起杆，再调整站距
- 蹲下去时加入关节扭矩，蹲起来再往外旋
- 大重量时可以用瓦氏呼吸，蹲下去之前先吸口气绷住，蹲起来再慢慢呼出
- 回杆要撞一下
